TRABAJO FRONTEND

Crear un dasboard en el que el estudiante podrá consultar información como
adeudos, informacion personal (nombre, matricula, etc), tutores, informacion socioeconomica.

El estudiante podrá realizar cambios en los datos del tutor, en caso de que no sea permitido
(opcion enable: False en la respuesta al principio), la informacion del tutor solo se mostrara
en un formulatio inhabilitado, lo mismo con la informacion socioeconomica, los estratos no se podrán
cambiar directamente por el estudiante, sino que serán el resultado de actualizar la informacion 
socioeconomica.

Antes de que la página esta cargada, envíen una peticion al endpoint donde el servidor le brindará atencion.
una peticion como esta:


const response = await fetch("http://127.0.0.1:8000/students/info", {
    headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Barer <Aqui va el token access, lo encuentran en las cookies>'
    },
});

const data = await response.json();
